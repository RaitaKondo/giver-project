# input_prompt

## ユーザー依頼の原文
プロフィール編集でアカウント写真を変更できるようにしたい。ユーザーが写真をフォルダから選択する。投稿の写真と同じようにGCSを使う予定です。写真の選択は一枚。容量も規定します。
何が必要か教えてください

## この作業で扱う対象
- プロフィール編集におけるアカウント写真変更機能の要件整理
- フロントエンド・バックエンド・GCS連携・バリデーション・運用観点の必要事項整理

## 前提条件
- リポジトリは `backend` / `frontend` を含む構成である
- 画像保存先として Google Cloud Storage (GCS) を利用予定
- 画像選択は1枚のみ
- 画像サイズ上限を設ける

## 制約事項
- この時点では実装依頼ではなく、必要事項の整理が主目的
- 既存の投稿画像アップロード方式との整合性を重視

## 実施前の整理メモ
- 既存の投稿画像アップロード実装があるため、再利用可能な設計・実装要素を確認する
- 要件漏れが出やすい領域（認可、バリデーション、古い画像の削除、キャッシュ対策、障害対応）を重点確認する

## 実施方針
1. プロジェクト構成と既存アップロード関連情報を把握する
2. 必要事項を「仕様」「API」「セキュリティ」「運用」「テスト」に分解して整理する
3. すぐ着手できる実装タスクに落とし込んで提示する
